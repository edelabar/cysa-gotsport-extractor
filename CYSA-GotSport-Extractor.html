CYSA GotSport Extractor

Choose a GotSport registrations CSV. This tool runs entirely in this browser—no file is uploaded anywhere.


How to get the GotSport CSV:⌄
  1. Select Programs > Program Registrations from the left menu.
  2. Set Program to the current season’s Rec Program (for example, CYSA U6 & Rec Fall 2026).
  3. Make sure Checkout Status is Completed and Submitted.
  4. Click Search.
  5. Above the results, click the button with three horizontal lines on the right side of the screen.
  6. Choose Download as CSV.
  7. Drag the downloaded file into the Drop the registrations CSV here box below.
